# bed-light-strip

## Interface

- Joystick used for brightness on Y-axis and hue on X-axis.

- Button one for on/off.

- Button two for effects e.g. number of colors, fade in and out, move, randomly fade in and out
    - When on an effect have joystick X-axis be used for changing the effect e.g. time of fade, speed of move

- Button three for keep current effect even when pressing the effects button

- Button four for disallow button presses
